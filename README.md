# trolley

A simple package tool for Slackware. Designed for unRAID.

# Install

    wget --no-check-certificate https://github.com/nicinabox/trolley/releases/download/0.1.1/trolley-0.1.1.txz
    installpkg trolley*

# Usage

    trolley search NAME
    trolley install NAME [VERSION]
    trolley remove NAME
